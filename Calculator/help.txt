COMMAND LIST
-------------

General
--------
1. Type "exit" to close the program.
2. "Whitespace" is ignored in all calculations. So, type it how you want.

Basic Math (only STANDARD CALCULATION MODE)
-----------
1. Number + Number
2. Number - Number
3. Number / Number
4. Number * Number
5. Number ^ Number
6. Number % Number (Modulo)

Advanced Math
--------------
1. derive 'Function' (only STANDARD CALCULATION MODE)
	1.1 derive x			
	1.2 derive constant number 	(Example: 'derive 3')
	1.3 derive x^(n)	   	(Example: 'derive x^(3)')
	1.4 derive a*x^(n)	   	(Example: 'derive 3*x^(3)')
	1.5 derive a*x^(n) + a*x^(n)	(Example: 'derive 3*x^(3) + 3*x^(3)')
	1.6 derive a*x^(n) * a*x^(n)	(Example: 'derive 3*x^(3) * 3*x^(3)')

2. rnd (Creates random Numbers. Acts as number. Only STANDARD CALCULATION MODE) 
3. mat (Matrix Calculations. Only STANDARD CALCULATION MODE)
4. sum (Summation Sigma Notation. "Number sum Number")
